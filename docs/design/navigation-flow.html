评分算法

Vision 框架 · 规则评分 · 无需外部模型

维度 权重 实现方式
人脸 / 宝宝置信度 30% VNDetectFaceRectanglesRequest · 检测是否有人脸、人脸大小占比Vision
清晰度 25% 图像拉普拉斯方差（缩略图计算），严重模糊直接过滤
表情 / 眼睛状态 20% VNDetectFaceLandmarksRequest · 眼睛开合、嘴角上扬Vision
构图完整度 15% 人脸不超出画面边缘、主体居中比例
场景意义 10% 连拍分组（拍摄时间差 < 2s）中非最高分照片降权
重复 / 连拍惩罚 扣分 同组最多保留 2 张，其余按分差递减
扫描流程（7 步）
1. 读取最近 90 天 或 1000 张（取并集上限）
2. 过滤截图、二维码、视频、时长 > 0 的资源
3. 分批（每批 50 张）加载缩略图，计算清晰度，丢弃方差 < 阈值的照片
4. 对剩余照片跑 VNDetectFaceRectanglesRequest，过滤无人脸
5. 对候选照片跑 VNDetectFaceLandmarksRequest，获取表情分
6. 按拍摄时间差做连拍分组，组内限制保留数量
7. 综合评分 → 排序 → 取前 20
