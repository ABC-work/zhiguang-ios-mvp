# 稚光 iOS MVP

稚光是一款面向宝宝家庭的本地 AI 照片精选与成长成册工具。

它不是全量云相册，第一版只验证一个核心闭环：

> 用户授权相册后，App 在本地扫描最近 1000 张照片，自动挑出宝宝照片中质量最高的 20 张，用户可手动调整，并一键保存到系统相册「稚光精选」。

## 仓库内容

- `product/index.html`：产品展示页，包含差异化定位、能力地图、核心流程、交互逻辑和 PRD 草案。
- `docs/ios-mvp-spec.md`：iOS MVP 开发规格，包含页面结构、数据结构、算法方案、异常状态和验收标准。
- `HANDOFF.md`：后续开发交接说明。

## 产品差异化

- 不做全量云备份，先做少量高价值照片精选。
- 不按日期简单堆叠照片，而是按宝宝成长节点组织内容。
- 不做通用共享链接，优先设计长辈友好的家庭分享。
- 默认本地处理，照片不默认上传。

## 建议技术路线

第一版建议用 Expo / React Native 快速验证页面和闭环。

如果后续要做更强的本地识别性能，再补 iOS 原生模块或迁移 SwiftUI + PhotoKit + Vision。
